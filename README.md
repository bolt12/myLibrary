# myPrelude
A not so naive implementation attempt of some of the pre defined functions in Haskell's Prelude and more.

# List of functions implemented
- enumFromTo, 
- enumFromThenTo, 
- concat, 
- last, 
- init, 
- (!!), 
- reverse, 
- take, 
- drop, 
- zip,
- elem,
- replicate,
- intersperce,
- group,
- inits,
- tails,
- isPrefixOf,
- isSuffixOf,
- isSubsequenceOf,
- elemIndices,
- nub,
- delete,
- (\\)
